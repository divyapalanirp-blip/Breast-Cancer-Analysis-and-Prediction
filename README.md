# Breast Cancer Analysis and Prediction

## Project Objective
This project analyzes a real-world breast cancer dataset and uses machine learning to predict whether a tumor is benign or malignant.

## Dataset
Breast Cancer Dataset

## Tools and Libraries Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

## Project Workflow

### 1. Data Analysis
- Loaded and explored the dataset
- Checked data types and statistical summaries
- Verified missing values

### 2. Data Visualization
- Diagnosis Distribution
- Radius Mean Distribution
- Texture Mean Distribution
- Correlation Heatmap

### 3. Machine Learning
- Encoded target variable (Diagnosis)
- Split data into training and testing sets
- Applied Random Forest Classifier
- Predicted tumor diagnosis

### 4. Model Evaluation
- Accuracy Score
- Confusion Matrix

## Results
The model successfully classified breast cancer cases using medical measurements and achieved high prediction accuracy.

## Conclusion
This project demonstrates the application of data science and machine learning in healthcare for breast cancer prediction and analysis.
